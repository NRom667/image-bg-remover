# 背景リムーバー
このアプリは、画像の背景を削除し、透過pngとして保存するためのアプリです。


# 特徴
- 前景点 / 背景点を追加し、切り抜き箇所を細かく調整可能
  - 細かい調整が不要なケースでは、windows付属のペイントが便利です
- SAM2.1による高精度なセグメンテーション
  - アプリ内でSAM2.1のダウンロードが可能


# 機能詳細
- jpg/ pngを読込
- 前景点 / 背景点を追加
- モデル選択：SAM2.1の tiny, small, base+, large
- マスクのフチをぼかすかどうか、ぼかし度合いの調整
- 透過pngとして保存

# 対象環境
- windows 11

# 使用している技術
- python
- PySide6 : GUI描画
- meta/SAM2.1 : 画像のセグメンテーション
  - CPU処理のみ対応

# AIの使用について
このアプリの作成にあたっては、codexを活用しています。

# ビルド方法・コマンド等メモ
- ビルドせず直実行する場合: python main.py
- build：scripts\build_windows.ps1を実行
- バージョン情報：src\__init__.pyに記載
